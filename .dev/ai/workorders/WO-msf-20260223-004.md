## Request

Implement build-time citation validation so unresolved citation IDs, orphan records, and broken source references fail before deployment.

## Scope

1. Build citation lint script.
2. Hook validation into build pipeline.
3. Emit actionable lint output.

## Tasks

1. Create `scripts/validate-citations.cjs`.
2. Validate all citation IDs against `sources.json`.
3. Detect orphaned sources and invalid backlink references.
4. Update npm scripts/build flow to run validation pre-build or post-sync.

## Success Criteria

- Invalid citations break build with clear error messages.
- Orphan and mismatch reports are generated deterministically.
- Local and CI build behavior are aligned.
